 
# Changelog

## [1.0.0] - 2026-01-07
### Added
- Initial release of QtQuick Template Project with AI-assisted workflows
- Modular CMake-based build system
- QtQuick 2.15 UI using MVVM pattern
- C++17 backend logic (JoystickViewModel reference implementation)
- Example QML and C++ unit tests
- Cross-platform support: Desktop and WebAssembly
- Stepwise requirements analysis pipeline (Markdown/YAML)
- Coding and testing guidelines in `copilot/`
- CI workflow for build and test (GitHub Actions)
- Comprehensive documentation and template usage instructions
